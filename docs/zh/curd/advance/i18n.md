# 国际化

CURD 组件支持国际化配置,可以自定义各种文案。

## 配置方式

```ts
import { createCurdConfig } from '@uozi-admin/curd'

app.use(createCurdConfig({
  locale: {
    // 设置语言
    locale: 'zh-CN',

    // 自定义文案
    messages: {
      'zh-CN': {
        add: '新增',
        edit: '编辑',
        delete: '删除',
        save: '保存',
        cancel: '取消',
        search: '搜索',
        reset: '重置',
        // ...
      },
      'en-US': {
        add: 'Add',
        edit: 'Edit',
        delete: 'Delete',
        save: 'Save',
        cancel: 'Cancel',
        search: 'Search',
        reset: 'Reset',
        // ...
      }
    }
  }
}))
```

## 切换语言

```ts
import { useLocale } from '@uozi-admin/curd'

const { setLocale } = useLocale()

// 切换语言
setLocale('en-US')
```

## 扩展语言包

```ts
import { extendMessages } from '@uozi-admin/curd'

extendMessages({
  'zh-CN': {
    custom: '自定义文案'
  },
  'en-US': {
    custom: 'Custom Text'
  }
})
```

## 组件内使用

```vue
<script setup>
import { useLocale } from '@uozi-admin/curd'

const { t } = useLocale()
</script>

<template>
  <button>{{ t('custom') }}</button>
</template>
```
